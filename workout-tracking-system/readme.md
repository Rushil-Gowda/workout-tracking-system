# 💪 Workout Tracking System

A full-stack workout tracking application designed to help users log workouts, track progress, and analyze strength performance over time.

---

## 🚀 Features

* User Authentication (Signup/Login)
* Create and manage workout routines
* Track sets, reps, and weights per session
* Strength analytics and progress graphs
* AI-style insights for performance improvement
* Previous workout memory + progression tracking
* Dark mode support
* Session-based tracking (date + time)

---

## 🧠 Tech Stack

Frontend:

* React (TypeScript)
* Tailwind CSS

Backend:

* Node.js + Express

---

## ⚙️ Setup Instructions

Clone the repository:

git clone https://github.com/Rushil-Gowda/workout-tracking-system
cd workout-tracking-system

Install dependencies:

npm install

Run frontend:

npm run dev

Run backend (if separate):

cd backend
npm install
node server.js
